`#` forms a comment

`"""` at the end and beginning forms multi-line comments

```python
# This is a simple hello world Python program
print("Hello World")

""" 
This program has a simple text print that concatenates two text strings Hello and 
name
"""
Hello = "Hello"
name = "Elian"
print(Hello, ", ", name)
```
